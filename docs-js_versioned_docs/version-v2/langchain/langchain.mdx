The `@sap-ai-sdk/langchain` package provides LangChain clients built on top of the foundation models and orchestration clients of the SAP Cloud SDK for AI.

Available clients:

    - `OrchestrationClient`
    - `AzureOpenAiChatClient`
    - `AzureOpenAiEmbeddingClient`

## Installation

```
$ npm install @sap-ai-sdk/langchain
```

:::important
Use the same `@langchain/core` version as the `@sap-ai-sdk/langchain` package.
Check the [`package.json`](https://github.com/SAP/ai-sdk-js/blob/main/packages/langchain/package.json) of the `@sap-ai-sdk/langchain` package to find the version in use.
:::
